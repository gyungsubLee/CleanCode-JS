# 명시적인 연산자 사용 지양하기

[연산자 우선 순위](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Operators/Operator_Precedence)

연산자 사용 시 `()`로 담아서 사용

```javascript
let number;

// 예측 가능하고 디버깅 하기 쉽게 작성
//  number--; -> number = number - 1;
//연산자 우선 순위 ((X + Y) * Z)


function decrement() {
    // number--;

    number = number - 1;
}

function increment()
    // number++;

    number = number + 1;
```
