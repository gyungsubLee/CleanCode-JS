# Prefix(접두사), Suffix(접미사)
> ```Prefiex```: 접두사, 단어 앞에 추가되어 새로운 단어를 만듬.<br/>
ex) 'un' + 'happy' = 'unhappy'<br/>
>
> ```Suffix```: 접미사, 단어 뒤에 추가 ... <br/>
> ex)  '-s' : 복수

## 정리 

Prefix(접두사), Suffix(접미사) convention을 통해 공통적인 속성 및 더 구체적인 표현을 통해 명시적으로 코드 작성 가능하다.
